JavaScript Course Assignment

Learn and explore JavaScript objects

Reminder: This assignment is due by Friday, 5 PM. Make sure to submit it on time!

Assignment Instructions

Write a short essay explaining what JavaScript objects are, how they are created, and their importance in programming. Include examples of object literals and constructors.

Create a JavaScript program that defines an object representing a student. The object should include properties like name, age, and courses (as an array). Add methods to display the student's details and calculate the total number of courses.

Extend the student object to include a nested object for grades. Write a method to calculate the average grade and display it.
